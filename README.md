# Java Application using Gradle

## Overview
This project demonstrates how to build a Java application using Gradle. It includes dependency management, automated builds, and a GitHub Actions CI workflow.

## Features
- Java application built with Gradle
- Dependency management using Gradle
- Automated build process
- GitHub Actions CI pipeline

## Technologies Used
- Java 17
- Gradle
- GitHub Actions

## Project Structure
```
java-gradle-demo/
├── app/
├── gradle/
├── .github/workflows/
├── gradlew
├── gradlew.bat
└── settings.gradle
```

## How to Build

```bash
./gradlew build
```

## How to Run

```bash
./gradlew run
```

## CI/CD

A GitHub Actions workflow automatically builds the project whenever code is pushed to the `main` branch.
